# Discord Checker

A Discord account checker tool with a simple web interface.

## Features

- Check Discord accounts
- Simple and clean web interface
- Progress tracking with JSON file
- Easy to use

## Project Structure

```
discord_checker/
├── app/
│   ├── checker.py      # Core checking logic
│   ├── main.py         # Main application entry
│   └── utils.py        # Utility functions
├── static/
│   └── frontend.html   # Web interface
└── result/
    └── progress.json   # Progress tracking data
```

## Requirements

- Python 3.x
- Required Python packages (install with `pip install -r requirements.txt`)

## Usage

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

2. Run the application:
   ```bash
   python app/main.py
   ```

3. Open your browser and navigate to the web interface URL.
